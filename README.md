#Auto-Ebook-Kindle-Uploader

Add a book to your Kindle by selecting the book file through the file dialog box.
It automatically detects your Kindle, so you just need to plug it to the computer.

Example use:
```bash
Welcome to Auto Ebook Kindle Uploader!
1. Add a book
2. Exit
Enter choice: 1
#(You select the book you want to add)
Book added to Kindle.
Enter choice: 2
Bye!
```

Tested only on Kindle Paperwhite 11 Generation and Windows 10/11.
